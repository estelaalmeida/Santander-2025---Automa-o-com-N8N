**Instalando o N8N** Não open sourcer (Pago)
Exite 3 maneira de instalação, vai de acordo com o ambiente.

_Local setup_
Desktop: vatagem, Não paga servidor de nuvem. não utilizar. Sem suporte!
Desvantagem o computador torna o servidor, não pode feichar o programa.Ferrametas podem não funcionar da melhro forma.

Node plataform: programa , é instalado uma instancia  requer conhecimento de node
Docker inatall, ferramentaas para gerenciar conteiner é instalado uma instacia do N8N.

_N8N CLOUD_ (Recomendada) : Utilizar o N8N dentro da plataforma;
Desvatagem:  pago.

_HOST SERVICE_ (Meio termo) : você hospeda o 8N8, o programa é de forma gratuita, porém vc é responsável pela hospedagem.Ex. Railway. Requer configuração

Passo a passo:
* nodejs.org;
* npm instal n8n -g
* n8n start
